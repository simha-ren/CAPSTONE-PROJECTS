# NexaBank GenAI Use Case Assessment

## Overview

This project evaluates multiple business challenges at NexaBank and maps them to the most suitable Generative AI (GenAI) capabilities, recommended models, business domains, and expected business outcomes.

The rule engine automatically classifies the use case, recommends an AI model, estimates ROI, and suggests extension tasks.

---

# Use Case A: Multilingual Email Triage

## Business Challenge

- 8,000 customer emails arrive daily.
- Emails are received in 12 different languages.
- Agents spend approximately 7 minutes manually categorizing each email.

## AI Classification

| Attribute | Value |
|------------|---------|
| GenAI Type | Text Generation |
| Recommended Model | Claude 3.5 Sonnet |
| Alternative Model | GPT-4o |
| Business Domain | Customer Service |

## Expected ROI

| Metric | Impact |
|----------|---------|
| Time Saved | 68% |
| Cost Reduction | £320k/year |
| Quality Lift | +22% |

## Extension Tasks

- Language detection
- Email summarization
- Sentiment analysis
- Priority scoring
- Automatic routing
- Auto-generated responses
