# GAN for MNIST Image Generation

This notebook implements a Generative Adversarial Network (GAN) to generate synthetic images resembling the MNIST handwritten digits.

## Introduction to GANs

Generative Adversarial Networks (GANs) are a class of artificial intelligence algorithms used in unsupervised machine learning, implemented by a system of two neural networks contesting with each other in a zero-sum game framework.

*   **Generator (G)**: Learns to create plausible data. Its input is random noise, and its output is data (e.g., images).
*   **Discriminator (D)**: Learns to distinguish between real data and data created by the generator.

The training process involves a continuous adversarial game where the Generator tries to fool the Discriminator, and the Discriminator tries to correctly identify real vs. fake data.

## Model Architecture

### Generator

The Generator (`Generator` class) is a deep convolutional neural network that takes a latent space vector (random noise `z_dim`) as input and upsamples it through a series of `ConvTranspose2d` (deconvolutional) layers to produce a 28x28 grayscale image.

*   **Input**: `z_dim` (100-dimensional random vector), reshaped to 1x1x`z_dim`.
*   **Layers**: 
    *   `ConvTranspose2d`: `z_dim` -> 256 channels, 7x7 output
    *   `ConvTranspose2d`: 256 -> 128 channels, 14x14 output
    *   `ConvTranspose2d`: 128 -> 1 channel, 28x28 output (grayscale image)
*   **Activation**: `ReLU` for intermediate layers, `Tanh` for the final output layer (to normalize pixel values between -1 and 1).

### Discriminator

The Discriminator (`Discriminator` class) is a deep convolutional neural network that takes a 28x28 grayscale image as input and classifies it as either "real" (from the MNIST dataset) or "fake" (generated by the Generator).

*   **Input**: 1-channel, 28x28 grayscale image.
*   **Layers**: 
    *   `Conv2d`: 1 -> 64 channels, 14x14 output
    *   `Conv2d`: 64 -> 128 channels, 7x7 output
    *   `Flatten`: Flattens the 128x7x7 feature map.
    *   `Linear`: Maps to a single output neuron.
*   **Activation**: `LeakyReLU` for convolutional layers, `Sigmoid` for the final output layer (to produce a probability score between 0 and 1).

## Training

The GAN is trained using the following hyperparameters:

*   **Batch Size**: 128
*   **Learning Rate**: 0.002
*   **Latent Dimension (`z_dim`)**: 100
*   **Epochs**: 10

The training process involves:

1.  **Discriminator Training**:
    *   Generate `fake` images from random `noise` using the Generator.
    *   Calculate loss for `real` MNIST images (target: 0.9 for label smoothing).
    *   Calculate loss for `fake` images (target: 0).
    *   Combine losses and update Discriminator weights.
2.  **Generator Training**:
    *   Generate `fake` images from random `noise`.
    *   Calculate loss, trying to make the Discriminator classify these `fake` images as `real` (target: 0.9).
    *   Update Generator weights.
