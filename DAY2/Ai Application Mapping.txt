| AI Application                      | Model Type                                                              | Primary Metric                            | Why This Metric?                                                                                                                                        |
| ----------------------------------- | ----------------------------------------------------------------------- | ----------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Anomaly Detection in Invoices       | Unsupervised Learning (Isolation Forest, Autoencoder, One-Class SVM)    | **Precision@K**                           | Auditors investigate only a limited number of flagged invoices. High precision ensures most alerts are truly suspicious, reducing investigation effort. |
| Spam Classification in Audit Emails | Supervised Classification (Logistic Regression, Random Forest, BERT)    | **F1 Score**                              | Balances false positives (important emails marked as spam) and false negatives (spam reaching auditors). Both errors are costly.                        |
| Predicting Deal Close Probability   | Supervised Classification (Gradient Boosting, XGBoost, Neural Networks) | **ROC-AUC**                               | Measures how well the model distinguishes deals that will close from those that won't across different probability thresholds.                          |
| Summarizing 100-Page Contracts      | Large Language Model (Transformer-based NLP, GPT, Claude, Llama)        | **ROUGE Score** (or Human Evaluation)     | Evaluates how well the summary captures key information from the original contract. Human review is often used for legal accuracy.                      |
| Translating Audit Reports           | Neural Machine Translation (Transformer Models)                         | **BLEU Score**                            | Measures translation quality by comparing generated text with expert translations while preserving meaning and terminology.                             |
| Flagging Revenue Forecasting Errors | Time-Series Forecasting + Anomaly Detection (Prophet, LSTM, XGBoost)    | **Mean Absolute Percentage Error (MAPE)** | Quantifies forecast accuracy in percentage terms, making it easy for finance teams to assess forecasting performance.                                   |
